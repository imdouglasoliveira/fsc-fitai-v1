# ❌ Erros Comuns

## Erro: "Repository not found"

**Problema:**
```bash
$ git clone git@github.com:adrianomorais-ganbatte/repo.git
ERROR: Repository not found.
```

**Solução:** Use o ALIAS, não `github.com`
```bash
# ❌ ERRADO
git clone git@github.com:adrianomorais-ganbatte/repo.git

# ✅ CORRETO
git clone git@github-adriano:adrianomorais-ganbatte/repo.git
```

---

## Erro: "Permission denied (publickey)"

**Soluções:**
```bash
# 1. Verificar e carregar chave
ssh-add -l
ssh-add ~/.ssh/id_ed25519_adriano

# 2. Verificar configuração
cat ~/.ssh/config | grep -A 5 "github-adriano"

# 3. Testar conexão
ssh -T git@github-adriano
```

---

## Erro: "Could not resolve hostname github-adriano"

**Problema:** Alias não está no `~/.ssh/config`

**Solução:**
```bash
# Verificar se existe
cat ~/.ssh/config | grep "github-adriano"

# Se não existir, adicionar manualmente
```
