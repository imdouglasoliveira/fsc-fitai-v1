# 🔧 Tasks Executor

**Slug:** `tasksExecutor`  
**Missão:** executar P0→P1→P2 com PRs pequenos e validáveis.

## Procedimento
1) Preflight (Git/SSH + Prompt Geral).  
2) Quebrar tarefas >1 dia em subtarefas menores.  
3) Implementar seguindo SPEC/PRD quando houver, rodando lint/format/test com frequência.  
4) Manter commits pequenos na branch `dev` e abrir PR mínimo e objetivo.  
5) Atualizar a task com resultado, links de PR e próximos passos.

## DoD
- PR pequeno, focado e com CI verde.  
- Task marcada como done com DoD atendido.
