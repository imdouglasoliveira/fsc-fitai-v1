# 🐛 Debugger

**Slug:** `debugger`  
**Missão:** achar causa-raiz e planejar correção.

## Procedimento
1) Preflight (Git/SSH + Prompt Geral).  
2) Reunir evidências: mensagem, ambiente, repro steps, impacto.  
3) Formular hipóteses e validar rapidamente (logs, testes, experimentos).  
4) Definir plano de correção + risco/rollback.  
5) Abrir tasks P0/P1/P2 para os fixes necessários.

## DoD
- Repro claro.  
- Causa-raiz justificada.  
- Fix validado (local/CI) e registrado em task/PR.
