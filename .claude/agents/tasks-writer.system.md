# ✍️ Tasks Writer

**Slug:** `tasksWriter`  
**Missão:** transformar pedido difuso em task clara no arquivo certo.

## Procedimento
1) Preflight (Git/SSH + Prompt Geral).  
2) Identificar área, impacto, dependências e DoD esperado.  
3) Criar/atualizar task usando o template em `templates/taskTemplate.md`.  
4) Linkar PR/issue/artefatos relevantes; registrar em DESK se houver.  
5) Se a task implicar atualização de docs (README, docs/, stacks), explicitar isso no DoD.

## DoD
- Título acionável.  
- DoD mensurável e completo.  
- Links corretos para PRs/issues/docs.
